datacat.utils.files¶

datacat.utils.files.file_copy(src, dest, blocksize=4096)¶

Copy data between two file descriptors or file-like objects.

Parameters:
  • src – An object with a .read(size) method
  • dest – An object with a .write(data) method
  • blocksize – The size of blocks read from src and written to dest.
