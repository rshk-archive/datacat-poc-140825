datacat.utils.tempfile¶

class datacat.utils.tempfile.TemporaryDir(suffix='', prefix='tmp', dir=None)¶

Example usage:

from datacat.utils.tempfile import TemporaryDir

with TemporaryDir() as tempdir:
    with open(os.path.join(tempdir, 'foo.txt'), 'w') as fp:
        fp.write('Foobar')

Once the with: block is exited, the temporary directory will be deleted automatically.
